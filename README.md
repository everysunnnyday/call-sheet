# 촬영 콜시트 템플릿 (Call Sheet Template)

촬영 콜시트를 **웹페이지**로 만들고, **링크 하나로 스태프에게 공유**하는 템플릿입니다.
서버·데이터베이스 없이 동작하며, 인터넷에 무료로 올릴 수 있습니다. (앱스토어 불필요)

---

## 1. 이게 어떤 방식인가요? (원리)

- **HTML 웹페이지**로 만들었습니다. → 링크만 있으면 카톡·문자·어떤 폰에서도 열립니다.
- **제작 페이지(`index.html`)** 에서 내용을 입력하고, **[링크 만들기·공유]** 를 누르면
  콜시트 내용이 통째로 **링크 안에 압축**되어 들어갑니다.
- 그 링크를 받은 사람은 **보기 페이지(`view.html`)** 에서 읽기 전용으로 봅니다.
- 즉, **서버에 저장하는 게 아니라 "링크 자체가 콜시트"** 입니다. → 비용 0원, 관리 필요 없음.

> 이 방식의 유일한 한계: 도면·자료 **이미지**는 링크에 담기 무거워서 1단계에선 제외했습니다.
> (자료는 "제목 + 외부 링크"로 넣습니다.) → 이미지 직접 첨부는 **2단계(Firebase)** 에서 추가합니다.

---

## 2. 파일 구성

| 파일 | 설명 |
|---|---|
| `index.html` | **제작 페이지** — 콜시트 입력·편집 (내가 사용) |
| `view.html` | **보기 페이지** — 링크로 공유되는 화면 (스태프가 열람) |
| `css/style.css` | 디자인 (반응형 + 인쇄용 A4) |
| `js/common.js` | 데이터 구조·샘플·링크 인코딩·콜시트 렌더 |
| `js/editor.js` | 제작 페이지 로직 |
| `js/viewer.js` | 보기 페이지 로직 (PNG·PDF·공유) |
| `js/lib/` | 라이브러리 3종 (압축·이미지·PDF 변환) |

---

## 3. 지금 바로 써보기 (내 컴퓨터에서)

이 폴더에서 아래 명령을 실행하면 미리보기 서버가 켜집니다.

```
python -m http.server 8777
```

그다음 브라우저에서 접속:
- 제작: `http://localhost:8777/index.html`
- 보기: 제작 페이지에서 [링크 만들기] 후 생성된 링크

---

## 4. 사용 방법

### 제작 (index.html)
1. **샘플 불러오기** 로 예시(코엑스 촬영)를 보고 구조를 익힙니다.
2. **새로 작성** 으로 비우고, 기본정보 → 담당자 → 콜 → 식사 → 자료 → 특이사항 → 타임테이블 순으로 입력.
3. 타임테이블은 **[＋ 행 추가]**, **[＋ 날짜 추가]** 로 얼마든지 늘릴 수 있습니다.
4. 입력값은 이 브라우저에 **자동 저장**됩니다. (창을 닫아도 유지)
5. 다 되면 **[🔗 링크 만들기·공유]** 클릭.
   - **스마트폰**: 카카오톡·메시지·기타 앱을 고르는 **공유창**이 바로 뜹니다.
   - **PC**: 링크가 **복사**됩니다. 붙여넣어 전달하세요.

### 보기 (view.html)
링크를 열면 콜시트가 보이고, 상단 버튼으로:
- **🔗 링크 공유** — 이 콜시트 링크를 다시 공유
- **🖼️ PNG** — 이미지로 저장/공유 (폰에선 카톡 등으로 바로 전송)
- **📄 PDF** — PDF로 저장/공유
- **🖨️ 인쇄** — 브라우저 인쇄. *"PDF로 저장"* 을 고르면 가장 **깨끗한 A4 PDF**가 나옵니다.
- **✏️ 편집** — 이 내용을 제작 페이지로 가져와 수정

> **꿀팁:** 가장 예쁜 PDF는 [🖨️ 인쇄] → 대상 "PDF로 저장" 입니다.
> [📄 PDF] 버튼은 폰에서 바로 공유할 때 편합니다.

## 5. 인터넷에 무료로 올리기 (GitHub Pages)

링크를 **다른 사람 폰에서도** 열려면 인터넷에 올려야 합니다. (localhost는 내 컴퓨터에서만 열림)

1. [github.com](https://github.com) 가입 후, 새 저장소(Repository) 생성 (예: `call-sheet`)
2. 이 폴더의 파일 전부를 저장소에 올립니다. (드래그 업로드 가능)
3. 저장소 → **Settings → Pages** → *Branch* 를 `main` / `(root)` 로 지정하고 저장
4. 잠시 뒤 `https://아이디.github.io/call-sheet/index.html` 주소가 열립니다.
5. 이제 이 주소가 **제작 페이지**입니다. 여기서 만든 링크를 누구에게나 공유하면 됩니다.

> 여러 사람이 각자 제작해도 됩니다. 각자의 콜시트는 각자의 링크에 담기므로 서로 안 섞이고, 비용도 계속 0원입니다.

---

## 6. 다음 단계 (2단계 예정 기능)

첨부해주신 요구사항 중 아래는 이미지 저장이 필요해 **2단계에서 Firebase 무료 등급**으로 확장합니다.

- 행사장 **도면 이미지** 업로드
- **타임테이블 ↔ 도면 연동**: 스케줄을 누르면 도면에 컬러로 위치 표시
- **자료(이미지·PDF) 직접 첨부** + 보기 화면에서 팝업 확대(닫기 X 버튼)

이때 링크는 지금처럼 길지 않고 `view.html?id=abc123` 형태로 짧아집니다.

## 7. 라이선스/라이브러리

- LZ-String (링크 압축) · html2canvas (이미지 변환) · jsPDF (PDF 변환) — 모두 오픈소스 무료
- 로컬에 내려받아 `js/lib/`에 포함 → 인터넷/CDN 상태와 무관하게 동작
